Añade un botón flotante de WhatsApp personalizable a tu sitio web de WordPress.

== Description ==
Este plugin añade un botón flotante de WhatsApp a tu sitio web de WordPress. Puedes personalizar la posición del botón, el número de WhatsApp al que se enlazará y el mensage previo.

== Installation ==
1. Sube la carpeta 'asu-whatsapp-floating-button' al directorio '/wp-content/plugins/'
2. Activa el plugin a través del menú 'Plugins' en WordPress
3. Configura el plugin en Ajustes > Botón WhatsApp

== Frequently Asked Questions ==
= ¿Puedo cambiar el icono del botón? =
Actualmente, el icono está integrado en el código. Para cambiarlo, necesitarías modificar el código del plugin.
